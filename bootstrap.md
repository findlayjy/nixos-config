# NixOS Bootstrap Guide

Steps to get a fresh machine running from `nixos-config`, using the graphical installer for the base install, then layering the flake config on top.

---

## 1. Install NixOS via the graphical installer

Boot from USB, run the graphical installer as normal, set up a username (`jamief`).

Install and reboot.

## 2. Log in and set up git identity

```bash
nix-shell -p git
git config --global user.name "Jamie Findlay"
git config --global user.email "jy.findlay@gmail.com"
```

## 3. Generate SSH key and add to GitHub

```bash
ssh-keygen -t ed25519 -C "jy.findlay@gmail.com"
cat ~/.ssh/id_ed25519.pub
```

Copy the output, add it at GitHub → Settings → SSH and GPG keys → New SSH key.

Test:

```bash
ssh -T git@github.com
```

## 4. Clone the config repos

```bash
git clone git@github.com:findlayjy/nixos-config.git ~/nixos-config
git clone git@github.com:findlayjy/.dotfiles.git ~/.dotfiles
```

## 5. Generate hardware config and copy it in

```bash
sudo nixos-generate-config # May not be necessary, but can't hurt
cp /etc/nixos/hardware-configuration.nix ~/nixos-config/hosts/pc/
```

## 6. Set `system.stateVersion` and `home.stateVersion`

Check what the installer's own `/etc/nixos/configuration.nix` set as `system.stateVersion` — copy that value into:

- `~/nixos-config/hosts/pc/configuration.nix`
- `~/nixos-config/hosts/pc/home.nix`

## 7. Rebuild using your flake

```bash
sudo nixos-rebuild switch --flake ~/nixos-config#{$HOST_NAME}
```

## 8. Reboot

```bash
sudo reboot
```

## Note on COSMIC setup

The COSMIC config gets autogenerated live, and trying to overwrite it while COSMIC is running can lead to issues. On first setup, it *should* work, since the default DE will be Gnome or something. But if you need to install the config while COSMIC is running, you have to drop into a TTY:

- `Ctrl`+`Alt`+`F2`, or, in the shell
```bash
sudo chvt 2
```

Once in the TTY, stop COSMIC
```bash
systemctl --user stop cosmic-session.target

```
Then `rm` the existing config and replace it:
```bash
rm -rf ~/.config/cosmic && ln -s ~/.dotfiles/.config/cosmic ~/.config/cosmic
```

To get back to the graphical session:
```bash
sudo chvt 1
```
